# Hylozoa Engine (Engine)

This is a C++ game engine made to handle complex environment just like the real world would.    
Think rainforest or the anthill in your garden.  

## Getting started

### Download

You can download the latest engine (pre)release [here](https://github.com/Julian52575/Hylozoa-Engine-Engine/releases).  
**But you probably want the [UI editor](https://github.com/Julian52575/Hylozoa-Engine/releases) to build your game using the engine.**  

### Collaborators
